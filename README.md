# GRC Security Portfolio

This repository contains hands-on Governance, Risk, and Compliance (GRC) projects demonstrating practical security auditing, compliance evaluation, and risk management.

The goal of this portfolio is to simulate real-world work performed by GRC analysts and IT security auditors.

---

# Project 1: ISO 27001 Compliance Assessment

## Overview

This project simulates an ISO 27001 compliance assessment for a fictional SaaS company named **NovaCloud AI Solutions**.

The assessment evaluates security controls against selected ISO/IEC 27001 Annex A requirements and identifies compliance gaps.

---

## Project Scope

The following security areas were evaluated:

- Information Security Governance
- Access Control
- Cryptography
- Logging and Monitoring
- Network Security
- Incident Management
- Regulatory Compliance

---

## Deliverables

• Company Security Profile  
• ISO 27001 Control Matrix  
• Evidence Documentation  
• Compliance Evaluation Report  

---

## Key Outcomes

- Evaluated 15 ISO 27001 Annex A controls
- Identified security control gaps
- Assigned control owners
- Documented audit evidence
- Produced compliance summary report

---

## Skills Demonstrated

- ISO 27001 Framework
- Compliance Gap Analysis
- Security Documentation
- Risk Identification
- Audit Evidence Review

---

## Tools Used

- Microsoft Excel
- Microsoft Word
- ISO 27001 Annex A Framework

# Project 2: Risk Register and Risk Assessment

## Overview

This project simulates a risk assessment for a SaaS organization by identifying, evaluating, and prioritizing security risks.

---

## Key Activities

- Identified 10 common security risks
- Calculated risk scores using likelihood and impact
- Categorized risks into High, Medium, and Low
- Defined mitigation strategies and ownership
- Created a risk summary report

---

## Skills Demonstrated

- Risk Assessment and Analysis
- Risk Scoring Methodology
- Risk Mitigation Planning
- Security Documentation
